# Support BeeVisible

Portail support et FAQ pour les clients BeeVisible — [support.beevisible.online](https://support.beevisible.online)

## Stack

- HTML / CSS / JS vanilla (compatible GoHighLevel)
- Font : Outfit + Montserrat (Google Fonts)
- Design system BeeVisible

## Structure

- `index.html` — Page principale (header, recherche, catégories, FAQ, formulaire, footer)
- Recherche instantanée dans la FAQ
- Accordéons expand/collapse
- Formulaire de contact avec validation
- Emplacement prévu pour widget Zendesk Chat

## Déploiement

Copier/coller le HTML dans une page GHL (mode code personnalisé) ou héberger en statique.
